Setup guide
1
Create a Google Cloud project
console.cloud.google.com → New Project → name it "Vida Ops" → select vidacayo.com organisation → Create
2
Enable Drive & Sheets APIs
APIs & Services → Library → "Google Drive API" → Enable. Repeat for "Google Sheets API".
3
Configure OAuth consent screen
APIs & Services → OAuth consent screen → Internal → App name: "Vida Ops" → add your email → Save
4
Create OAuth credentials
Credentials → + Create Credentials → OAuth client ID → Web application → Authorised JavaScript origins:
https://vidacayo.com
https://www.vidacayo.com
http://localhost
Copy the Client ID and paste it into Step 1 above.
